## 1. [Python for Java Developers: Basics](https://github.com/akashp1712/python-cheat-sheets/blob/master/Python_for_Java_developers_cheat_sheet.pdf)

- Basic Syntax
- Variables
- Data Types
- Operators
- Conditions
- Looping
- Iteration
